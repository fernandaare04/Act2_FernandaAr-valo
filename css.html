¿Qué es CSS?

CSS significa Cascading Style Sheets. Es un lenguaje que permite controlar el diseño visual de una página web.

Con CSS se pueden modificar diferentes aspectos de un sitio web:

  • Colores
  • Tipografía
  • Espacios entre elementos
  • Bordes y sombras
  • Distribución del contenido

Gracias a CSS los sitios web pueden ser más atractivos, ordenados y fáciles de utilizar.
